# apbd_and_tut3_task

This is console database project for university. This system allows you to add users, 
equipment and rent them manage those rents as well. 
To run the application simply run the main code, Program just demonstrates how it works.
I divided the project into 4 folder Users, Equipment, Services and UI so each class has it's
own resposibility. The logic of task is not put in one file App, but instead there are different
services handeling he logic. I did that services can work with each other, passing them to each other.
User, Equipment and Rental represent domain model
